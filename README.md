## What is this?

A weekly-updated, categorized list of **paid AI training jobs** — also called RLHF, data annotation, or AI feedback roles. These are legitimate remote freelance opportunities where you get paid to:

- Review and rank AI-generated responses
- Write prompts and code examples to train LLMs
- Label data, transcribe audio, or annotate images
- Evaluate outputs in your area of expertise (medicine, law, finance, coding, etc.)

All jobs include a referral link. Applying through the links here (via [aitrainer.work](https://aitrainer.work)) supports this project.
